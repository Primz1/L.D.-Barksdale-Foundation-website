## 💡 Popular Questions

### How do I change content?
Edit: `src/constants/content.ts`
Changes appear instantly on dev server!

### How do I change colors?
Edit: `tailwind.config.js`
Update the color definitions in the theme section.

### How do I add new images?
1. Place image in `/assets` folder
2. Import in component with: `import img from '../../assets/filename.png'`
3. Use in JSX: `<img src={img} />`

### How do I deploy online?
See: `DEPLOYMENT.md`
Options include Vercel, Netlify, or any static host.

### How do I test on mobile?
- Press `F12` in Chrome to open DevTools
- Click device icon and select mobile
- Or test on actual phone at your local IP

### What if something breaks?
Check: `DEVELOPMENT.md` → Troubleshooting section
